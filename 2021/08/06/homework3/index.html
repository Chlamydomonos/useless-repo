第三题 循环结构（一）

知识介绍

循环结构也是编程语言中一种十分重要的结构。有了循环结构，计算机才能一次性进行大量的运算。

在c++语言中，可以用for语句来表示循环结构。以下是一个示例：

可以看到，for语句后面的括号里有三个语句，之间用分号;分隔。for循环的执行过程如下：

  • 首先执行括号里的第一个语句
  • 之后，开始循环
    • 对于每一次循环，首先执行括号里的第二个语句，获取计算结果。
      • 如果结果为true
        • 先执行for循环之后的第一个语句
        • 再执行括号里的第三个语句
        • 继续循环
      • 如果结果为false
        • 结束循环，跳过for循环之后的第一个语句，执行之后的内容

因此，上面的循环语句的执行情况如下：

  • 首先执行int i = 0，定义变量i并赋初值0
  • 之后，开始循环
  • 第一次循环
    • 执行i < 3，结果为true
    • 执行cout << i << ' ';，输出i和一个空格
    • 执行i = i + 1，i的值变为1
  • 第二次循环
    • 执行i < 3，结果为true
    • 执行cout << i << ' ';，输出i和一个空格
    • 执行i = i + 1，i的值变为2
  • 第三次循环
    • 执行i < 3，结果为true
    • 执行cout << i << ' ';，输出i和一个空格
    • 执行i = i + 1，i的值变为3
  • 第四次循环
    • 执行i < 3，结果为false
    • 停止循环，跳过cout << i << ' ';，不输出
  • 正常执行cout << endl;
